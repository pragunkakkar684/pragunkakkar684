I’m someone who loves building things with code whether it’s a web app, a backend system, or just a fun little script to make life easier. I’m into Node.js, FastAPI, Python, and pretty much anything that lets me turn ideas into reality.

Right now, I’m working on Aarohan, a platform that simplifies real estate compliance. I love projects where I get to solve real problems and see my work actually make a difference.

GitHub is my playground. I experiment, learn, and share everything from web dev and automation to data tools and AI projects. I try to keep my code clean and understandable because I know someone else (or future me!) might look at it later.

Outside of coding, I’m the kind of person who’s always exploring new tech, geeking out over retro web design, or thinking of clever ways to solve problems. For me, coding isn’t just a skill it’s a way to create, explore, and have fun.
